# DSA Visualizer (Learning Journey)

This project started as a **Pathfinding Visualizer** and is evolving into a broader
**DSA learning workspace** where we implement and visualize algorithms from start to end.

## Vision

Build an interactive algorithm-learning app that helps us:

- understand algorithm behavior step-by-step,
- compare algorithm strategies visually,
- and grow from beginner to advanced DSA topics in one project.

## Current Status

### Implemented

- ✅ Breadth-First Search (BFS) on a 2D grid
	- Place Start / End
	- Draw walls
	- Animate visited cells
	- Animate shortest path
- ✅ Bubble Sort (Array Visualizer)
	- Generate random arrays
	- Animate comparisons and swaps
	- Highlight sorted portion/progress

### In Progress Direction

We are expanding this project in phases so it remains clean and easy to learn from.

## Learning Rules for This Repo

- Keep each algorithm in its own module.
- Keep visualization logic separate from algorithm logic.
- Add clear status messages and animations.
- Prefer readability over clever shortcuts.
- Build incrementally and test each step.
